# v0.6 Debug-Menu Interactive UAT Checklist

The v0.6 reviewer-feedback work (PR #158) wired a large debug menu — both the
ImGui **ribbon** and the **classic** SDL-text fallback — to a C action layer
that is unit-tested and CI-gated. What that automation **cannot** cover is the
actual UI wiring: that each button/slider/tab calls the right action and renders
without a GL-context crash. ImGui needs a real window, so it can't be driven
under `SDL_VIDEODRIVER=dummy`. This checklist is the manual pass that closes that
gap.

Three new items were added on top of PR #158 (this branch) and are folded in
below: a **Memory** watch tab, a **Hide cursor in fullscreen** toggle, and an
**Analog deadzone** slider.

## Already verified — do NOT re-test

These are covered by automated tests / headless runs; trust them:

- **C action layer** — `build/pc/debug_actions_test` (give-all, per-item toggle
  exclusivity, stat clamps, flags, dungeon items, bottles, figurines).
- **Memory-watch logic** — `build/pc/memory_watch_test` (little-endian reads,
  range boundary, unmapped-address safety, add/de-dup/remove/clear, 32-cap).
- **Colour correction / LCD persistence pixels** — eyeballed headless (see
  "Headless smoke" below). CC-on is correctly dimmer/warmer than CC-off;
  persistence ghosts moving elements only. No artifacts.
- **Boot + config load** — the binary boots, loads save/assets, and reads the
  two new config keys (`fullscreen_hide_cursor`, `analog_deadzone`) without
  error.

## Headless smoke (reproducible)

```sh
ROM=/home/sian/tmc/baserom.gba
COMMON="TMC_AUTOPLAY=1 TMC_BASEROM=$ROM TMC_NO_UPDATE_CHECK=1 SDL_VIDEODRIVER=dummy SDL_AUDIODRIVER=dummy"

# Colour-correction A/B (PNGs differ -> capture is post-CC):
env $COMMON TMC_COLOR_CORRECTION=1 TMC_CAPTURE_FRAME=300 TMC_CAPTURE_OUT=/tmp/cc_on.png  ./build/pc/tmc_pc --no-audio
env $COMMON TMC_COLOR_CORRECTION=0 TMC_CAPTURE_FRAME=300 TMC_CAPTURE_OUT=/tmp/cc_off.png ./build/pc/tmc_pc --no-audio
# LCD persistence:
env $COMMON TMC_LCD_PERSISTENCE=1 TMC_LCD_PERSISTENCE_RHO=0.6 TMC_CAPTURE_FRAME=300 TMC_CAPTURE_OUT=/tmp/persist.png ./build/pc/tmc_pc --no-audio
```

`exit 134` after `[capture] ... (ok=1)` is the capture harness's dump-then-bail
(threads torn down on hard exit); the PNG is written before it. Not a crash.

## Interactive pass — needs a real window

Launch normally, press **F8** to open the menu. The footer checkbox toggles
**Ribbon mode** vs **classic menu** — exercise both renderers for each section,
since they share the C layer but have independent UI code.

### NEW in this branch

- [ ] **Memory tab** (ribbon) / **Memory watch** page (classic)
  - [ ] Type a hex address (e.g. `03000000`), pick width (u8/u16/u32). The
        live preview next to the input shows a value (or `<unmapped>` for a bad
        address like `00000000`).
  - [ ] **Add watch** → it appears in the table; value updates every frame while
        the game runs.
  - [ ] Add the same address+width again → no duplicate row (de-dup).
  - [ ] Per-row **X** removes that row; remaining rows keep their values.
  - [ ] **Clear all** empties the list.
  - [ ] Classic menu: **Address** ◀▶ steps by **Step** (cycle the step size),
        **Width** cycles u8/u16/u32, **Add watch** appends, Enter on a watch row
        removes it.
  - [ ] Watch a known-changing value (e.g. Link's HP / rupee count after picking
        rupees) and confirm it tracks.
- [ ] **Display → Hide cursor in fullscreen** checkbox
  - [ ] Default ON. Go fullscreen (F11 / the Fullscreen checkbox) → OS cursor
        hidden.
  - [ ] Uncheck while fullscreen → cursor reappears immediately (live apply).
  - [ ] Re-check → cursor hides again. Setting persists across a restart.
  - [ ] Windowed mode always shows the cursor regardless of this setting.
- [ ] **Reborn → Analog deadzone** slider (only visible when **360° analog stick
      movement** is enabled)
  - [ ] Enable the 360° analog feature → the slider appears indented under it.
  - [ ] With a gamepad: raise the deadzone → small stick tilts no longer move
        Link; lower it → Link responds to lighter tilts. Default 0.30.
  - [ ] Value persists across a restart (`analog_deadzone` in config.json).
